# 基于ZendFramework3的脚手架

包含框架组件

        ZendFrameWork 3
        Vue 2.5+
        Vue-Lazyload 1.2.2
        Vue-resource 1.5.1
        Vue-router 3.0+
        Bootstrap 4.1+


使用
    
        git clone https://github.com/yharvey956/ZendFramwork-WebInit
		cd ZendFramwork-WebInit
			composer install
			composer dump-autoload
		配置nginx 访问public index.php
		打卡浏览器访问
